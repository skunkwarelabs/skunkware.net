Snapper

An app for fishing.

Source GPLv3

Data coverage

All data is bundled offline — no API keys, no runtime calls. Fishing regulations (bag/size limits, in-season status) ship for all 50 states. Fish stocking data is pulled per state from each state’s fish & wildlife agency, currently 24 states.
